# my-package

A simple npm package created on Android using Termux.

## Installation

~~~~sh
npm install my-package
~~~~

## Usage

~~~~js
const myPackage = require("my-package");

console.log(myPackage());
// → "Hello from my first npm package!"
~~~~

## Description

This package was built entirely on an Android tablet using Termux.  
It’s a minimal example showing how to create, publish, and use an npm module.
